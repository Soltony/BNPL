# **App Name**: BasePage

## Core Features:

- Simple display: Just display a home page with the name of the site and maybe the current time.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA), evokes a sense of simplicity and calm.
- Background color: Very light gray (#F5F5F5), almost white to provide a clean and unobtrusive backdrop.
- Accent color: Dusty Rose (#D8BFD8), a muted and gentle hue analogous to lavender, adds a subtle touch of elegance without overwhelming the minimalist design.
- Font: 'Inter', a sans-serif font, provides excellent readability and a modern feel suitable for body and headlines.
- A simple, centered layout ensures a focus on content and easy navigation, promoting clarity and ease of use.
